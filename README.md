# AI CodeDex Clone – EdTech Hackathon Project

An AI-powered beginner coding platform inspired by CodeCademy, designed to help users learn HTML/CSS, React..etc easy to hard langugues with personalized guidance, small exams, and saved progress.

---

## Core Features

### 1. AI Assistant (Powered by OpenRouter)
- Simple text input: "I want to learn HTML"
- Responds with friendly explanation + learning path
- Uses **OpenRouter API** to access powerful AI models

### 2. Interactive Lessons

- Small chunks: tags, elements, styling
- Code editor with **live preview**

### 3. Mini-Exams

- After 2–3 lessons
- Exam example: "Style a `container` class with flex and background"
- Auto-check user code for key patterns

### 4. Progress Tracking

- Save what user has completed
- Resume where user left off
- Show timeline: "You finished 3/10 lessons"
