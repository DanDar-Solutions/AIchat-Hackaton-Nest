# AI CodeDex Clone – EdTech Hackathon Project

## 🚀 Overview

An AI-powered beginner coding platform inspired by CodeCademy, designed to help users learn HTML/CSS, React..etc easy to hard langugues with personalized guidance, small exams, and saved progress. Built with modern UI/UX and using only free/open-source tools.

---

## 🎯 MVP Focus

- Teach **HTML & CSS basics, ReactJS+nodeJS**
- AI assistant guides users step-by-step
- Give interactive **mini-exams** after learning sections
- **Save progress** if users leave and come back
- Show **learning journey**: what the user has learned
- Simple, clean, beginner-friendly UI/UX

---

## 🧠 Core Features

### ✅ 1. AI Assistant (Powered by OpenRouter)
- Simple text input: "I want to learn HTML"
- Responds with friendly explanation + learning path
- Uses **OpenRouter API** to access powerful AI models

### ✅ 2. Interactive Lessons

- Small chunks: tags, elements, styling
- Code editor with **live preview**

### ✅ 3. Mini-Exams

- After 2–3 lessons
- Exam example: "Style a `container` class with flex and background"
- Auto-check user code for key patterns

### ✅ 4. Progress Tracking

- Save what user has completed
- Resume where user left off
- Show timeline: "You finished 3/10 lessons"

## 🔌 Setting up the Project

To run the AI assistant:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Install dependencies:
   ```
   cd Server
   npm install
   cd ../Client
   npm install
   ```

3. Start the server:
   ```
   cd Server
   npm run dev
   ```

4. Start the client:
   ```
   cd Client
   npm run dev
   ```

5. Access the application:
   - Open your browser and go to: http://localhost:5173
